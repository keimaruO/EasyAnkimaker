# EasyAnkimaker

推しの声で英単語を覚れるようにするツールです

Ankiっていう暗記カードアプリとYouTube再生可能にAnkiの拡張機能を利用します

さらにそれをホロメン歴代配信の発話検索使って問題の単語を喋ってるシーンを効率的に探して、問題を効率的に作れるようにしました。1問1分ぐらい？
